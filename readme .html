Company Website

A responsive and modern company website template built with HTML, CSS, and Bootstrap. The design focuses on simplicity, accessibility, and user engagement.

Preview

Below is a preview of the project:

Features

  • Responsive Navbar: A user-friendly navigation bar that adapts to different screen sizes.
  • Header Section: A visually appealing jumbotron to grab user attention.
  • Icon Section: Highlighted services with descriptive text and icons.
  • Contact Button: Easy call-to-action for visitors.
  • Footer: Simple and functional footer for branding.

Technologies Used

  • HTML5
  • CSS3 (custom styles in styling.css)
  • Bootstrap 3
  • Google Fonts (integrated for typography enhancements)

How to Use

  1. Clone or download the repository.
  2. Ensure the folder structure is as follows: 
    /images
|-- 1.png
|-- 2.png
|-- 3.png
|-- 4.png
|-- 5.jpg
|-- website-preview.png
/css
|-- bootstrap.min.css
|-- styling.css
/js
|-- bootstrap.min.js
index.html
  3. Open index.html in a browser to view the website.

Adding and Customizing Images

  • Navbar Logo: Update 4.png with your logo for the navigation bar.
  • Icons Section: Replace 1.png, 2.png, and 3.png with your custom icons for the "Price it," "Plan it," and "Make it" sections.
  • Background: Change 5.jpg to a new background image for the page.
  • Project Screenshot: Add a screenshot of your project as website-preview.png for use in this README.

Update the src attributes in index.html if you rename any images:

<img src="images/new-logo.png" class="logo">
